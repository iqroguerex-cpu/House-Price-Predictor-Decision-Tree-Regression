## 📌 Project Overview

This project predicts **house prices** based on features like:

* Area
* Number of bedrooms
* Bathrooms
* Stories
* Parking

It uses a **Decision Tree Regression model** and provides predictions through an **interactive dashboard built with Streamlit**.

---

## 🧠 How It Works (Simple Explanation)

Think of the model like a decision-making system:

> It learns from past house data and makes predictions for new houses.

Example:

* Bigger houses → higher prices
* More bedrooms → higher prices

The model finds these patterns automatically and uses them to estimate prices.
